# Arba- Full Stack Web App

# backendLink = https://arba-backend-2-0j6p.onrender.com/

## Tech Stack:

### Frontend:

- React.Js
- Chakra UI
- React-router-dom(v5)

### Backend:

- Node.js
- Express
- Mongoose
- Mongodb Atlas
- Cors
- dotenv
- bcrypt
- JSON Web Token

### Endpoints for frontend

- /login : Login user with creadentials
- /signup : register user
- / : Home page
